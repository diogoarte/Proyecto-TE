# Tienda Virtual con React

Este es un proyecto de una tienda virtual construida con **React**, **Vite**, **React Router**, **Zustand** y **CSS**. La tienda permite navegar entre productos, agregar al carrito, ver el detalle de cada producto y filtrar productos por categoría.

## Funcionalidades
- **Catálogo de productos**: Muestra una lista de productos con opción para ver más detalles.
- **Detalle del producto**: Muestra información completa del producto seleccionado.
- **Carrito de compras**: Permite agregar, eliminar y cambiar la cantidad de productos en el carrito.
- **Filtro por categoría**: Filtra los productos según la categoría seleccionada.
